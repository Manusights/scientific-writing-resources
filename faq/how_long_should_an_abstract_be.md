# How Long Should an Abstract Be

Follow journal limits first.

As a working rule, keep it as short as possible while still including problem, approach, main quantitative result, and one boundary statement.


## Rewrite drill
Take one paragraph, remove adjectives, add exact numbers, and keep one claim per sentence.
