# Abstract Writing Guide

A strong abstract does one job, it tells a busy reader exactly what changed because of this study.

## Structure that works
1. Problem: what gap is still open
2. Approach: model, cohort, or method in one sentence
3. Primary result: include numbers
4. Boundaries: what your result does not prove

## Quality check
- If you remove one sentence, does meaning collapse? If not, cut it.
- Replace adjectives with numbers.
- Keep causal language only when design supports it.
