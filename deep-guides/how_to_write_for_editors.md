# How to Write for Editors

Editors screen quickly. Your draft should make three things obvious in under one minute.

1. What changed because of this study
2. Why the change matters for this journal audience
3. Why the evidence is strong enough for review

## Practical tactics
- Put the core claim in plain language in the first paragraph
- Keep one main message per figure
- Use specific numbers in abstract and results
- State one key limitation before reviewers ask
