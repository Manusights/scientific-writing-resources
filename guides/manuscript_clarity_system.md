# Manuscript Clarity System

Clarity is mostly structure, not style polish.

## Core system
1. one claim per paragraph in results and discussion
2. one question per subsection
3. one figure purpose per figure
4. one limitation sentence per major conclusion

## Editing sequence
- pass 1: remove vague adjectives
- pass 2: add exact numbers
- pass 3: align claims with figures
- pass 4: tighten transitions and remove repetition

## Section-level targets
- Abstract: problem, method, main number, boundary
- Introduction: gap and objective
- Results: evidence first, interpretation later
- Discussion: meaning, limits, next step
